DataFrame Project

This is a project on making a Java library for tabular data manipulation with CSV support.

Features

  • CSV parsing
  • Data aggregation
  • Statistical operations

Convention

Naming Conventions

  • Use camelCase for method names and variables
  • Use PascalCase for class names

Developing branches convention

  • Use main for the main branch
  • Use dev for the development branch
  • Use feature/feature-name for feature branches
    e.g., feature/README
  • Use fix/bug-name for bug fix branches
  • Use perf/branch-name for performance improvements, refactors, or code cleanups

All feature, fix, and perf branches must start from dev. Once complete, create a pull request to dev for review and validation. Only after all validations can the feature be merged into the main branch.

How to run and compile

Use the following Maven command to compile and run the tests:

mvn clean install
